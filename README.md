PROJECT TITLE: LLM MISINFORMATION DETECTION


1. PROJECT OVERVIEW

This project focuses on the detection of misinformation using Large Language Models (LLMs). The core objective is to evaluate how well AI models can distinguish between real news and fake news. The study covers two types of misinformation: traditional fake news written by humans and synthetic misinformation generated by AI models.


2. DATASETS

This research utilizes two primary data sources:

FakeNewsNet: A collection of real-world news articles verified by human fact-checkers.

LLM-generated Misinformation: A dataset consisting of fake narratives created by various AI models to test the detection limits of our systems.


3. PROJECT STRUCTURE

data: Stores raw, processed, and synthetic data samples.

report: Contains the detailed research paper and documentation (paper.pdf).

results: Contains the evaluation outputs and JSON files for both models.

src: Includes the source code for data loading, dataset creation, and model evaluation.


4. MODELS AND METHODOLOGY

We implemented and compared two state-of-the-art Transformer models:

BERT (Bidirectional Encoder Representations from Transformers)

RoBERTa (Robustly Optimized BERT approach)

The methodology involves cleaning the text, fine-tuning the pre-trained models on a combined dataset of human and AI news, and measuring performance using standard NLP metrics.


5. PERFORMANCE RESULTS

The models were evaluated based on their accuracy in identifying misinformation:

Model       | Accuracy | F1-Score

BERT        | 0.83     | 0.82

RoBERTa     | 0.86     | 0.85

Observation: RoBERTa achieved better results than BERT, with an improvement of approximately 3% in both accuracy and F1-score. This indicates that RoBERTa is more effective at capturing the complex linguistic features of modern misinformation.


6. HOW TO RUN

Install required libraries: pip install -r requirements.txt

Prepare the final dataset: python src/create_final_dataset.py

Run the evaluation: python src/evaluate.py
