==================🚀 PROJECT: LLM MISINFORMATION DETECTION 🚀==================

1. 📋 PROJECT OVERVIEW

This project investigates the detection of misinformation using Large Language Models (LLMs). We compare BERT and RoBERTa to identify which model is more effective at catching fake news, whether it's written by humans or generated by AI.

2. 📊 DATASETS

   FakeNewsNet: Real-world news verified by humans. https://github.com/KaiDMML/FakeNewsNetv 

   LLM-generated: Synthetic fake news created by AI. https://github.com/llm-misinformation/llm-misinformation

5. 🤖 MODELS & METHODOLOGY

We utilize state-of-the-art Transformer architectures:

BERT: A baseline for context understanding.

RoBERTa: Optimized training for higher accuracy.
